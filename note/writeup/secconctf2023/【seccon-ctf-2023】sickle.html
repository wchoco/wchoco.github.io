内容としてはpickleでシリアライズした処理を読み込み、その結果をフラグの成否としている。
肝心のシリアライズされた部分で入力を受け取りフラグの判定をしていると思われる、シリアライズされた内容を確認する必要がある。

いくつかpickleでシリアライズされたものを見やすくデシリアライズしてくれるようなツールを試してみたが、上手くいかなかったのでpickletools[1]を使って命令列に戻した。
出力されるのはpickleの命令列であるので、pickleの実装を見たり、pickletoolsのannotate機能を利用するなどして読んでいく。

import pickletools

d = open("./pickeld.data", 'rb').read()

# STOP命令が出てくるたびにデシリアライズが止まってしまうので、
# そのあとから再度genopsを実行する。
out = []
for i in pickletools.genops(d):
    out.append(f"{i[0].name} {i[1]}")

for i in pickletools.genops(d[262:]):
    out.append(f"{i[0].name} {i[1]}")

for i in pickletools.genops(d[262+69:]):
    out.append(f"{i[0].name} {i[1]}")

open("./disass.md", "w").write("\n".join(out))

あとはこの処理を元に戻していけばよい。

p = 18446744073709551557
d = pow(65537, -1, p-1)
ns = [
    8215359690687096682,
    1862662588367509514,
    8350772864914849965,
    11616510986494699232,
    3711648467207374797,
    9722127090168848805,
    16780197523811627561,
    18138828537077112905
]
xor = 1244422970072434993

def print_str(n):
    while n > 0:
        print(chr(n & 0xff), end="")
        n = n >> 8

for n in ns:
    print_str(pow(n, d, p) ^ xor)
    xor = n
